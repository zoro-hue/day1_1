Welcome to MERN stack

MERN stack is a web development stack which is used to develop web application

About the page :-

This page is about MERN stack and its components. MERN stack is a combination of four technologies which are used to develop web application. MERN stack is a full stack development which means it can be used to develop both frontend and backend of a web application. MERN stack is a popular choice for web development because it is easy to learn and use. MERN stack is also a open source which means it is free to use and modify. MERN stack is also a cross platform which means it can be used to develop web application for different platforms like windows, mac, linux etc.

Frontend :

Frontend is the part of a web application which is visible to the user. Frontend is also known as client side. Frontend is developed using html, css and javascript. Frontend is responsible for the user interface and user experience of a web application. Frontend is also responsible for the communication with the backend of a web application.

  1. React js
  2. Node js

Backend :

Backend is the part of a web application which is not visible to the user. Backend is also known as server side. Backend is developed using Node js, Express js and Mongo db. Backend is responsible for the logic and functionality of a web application. Backend is also responsible for the communication with the database of a web application.

  • Express js
  • Mongo db
Technology Type
